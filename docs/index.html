Calibrate Against Reality

If you call a hospital and confirm their actual bed/ICU count, log it here. This builds a real, timestamped dataset that can eventually replace the simulated model, that's the honest path to accurate predictions, not longer training on synthetic data.

How This Works

Real: hospital locations, types, specialties & published bed/ICU counts (where verified); Bengaluru's major road corridors and known chronic bottlenecks (Silk Board, KR Puram, Marathahalli, Hebbal, Majestic); Karnataka's real 108/112 emergency numbers.

Simulated/Estimated: minute-to-minute hospital occupancy and (unless you add a Google Maps key) road congestion — no public live feed exists for either in Bengaluru today. Two deep learning models (an LSTM capacity forecaster and a Graph Neural Network traffic model) demonstrate what a production system fed by real data could do. Every occupancy figure in this app is labeled EST for exactly this reason.
